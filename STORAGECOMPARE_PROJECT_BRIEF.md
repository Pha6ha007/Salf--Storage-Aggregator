## 1. ПРОЕКТ В ОДНОМ АБЗАЦЕ

**StorageCompare.ae** — первый в ОАЭ агрегатор складских помещений для частных лиц и бизнеса. Платформа объединяет владельцев складов (операторов) и людей, которым нужно хранение вещей, в едином маркетплейсе с поиском на карте, фильтрацией, AI-рекомендациями размера бокса и онлайн-бронированием. По сути — «Bayut для складов»: пользователь находит склад, выбирает бокс, бронирует онлайн, оператор подтверждает.

---

## 2. ПРОБЛЕМА

### Что сейчас не так на рынке хранения в ОАЭ

- **Нет единой платформы.** Человек, которому нужен склад, гуглит, звонит по 5–10 номерам, ездит смотреть лично. Это занимает дни.
- **Непрозрачные цены.** Операторы не публикуют цены — узнать стоимость можно только по звонку. Сравнить невозможно.
- **Сложно выбрать размер.** «Мне нужно хранить диван, 20 коробок и велосипед — какой бокс нужен?» Без опыта невозможно ответить.
- **Операторы теряют клиентов.** У мелких и средних складов нет маркетинга. Они зависят от сарафана и Google Maps. Боксы простаивают при спросе на рынке.
- **Нет цифровизации.** Большинство операторов ведут учёт в Excel или на бумаге. Бронирование — по телефону. Аналитики нет.

### Масштаб проблемы

- Рынок self-storage ОАЭ растёт: экспаты, бизнес, e-commerce — спрос стабильный.
- Ни одного агрегатора с бронированием на рынке ОАЭ нет. Есть каталоги без функционала (Google Maps, справочники) или корпоративные решения.
- Ближайшие аналоги (SpareFoot, Neighbor в США) показали, что модель агрегатора работает и масштабируется.

## 3. РЕШЕНИЕ

### Что делает StorageCompare

| Функция | Для кого | Как работает |
|---------|----------|-------------|
| **Поиск на карте** | Пользователь | Ищет склады рядом с нужным местом (район, адрес). Карта + список. Фильтры: цена, размер, климат-контроль, охрана, 24/7 доступ |
| **Каталог с фильтрами** | Пользователь | Все склады в одном месте. Сортировка по цене, расстоянию, рейтингу. Фотографии, описание, характеристики |
| **AI Box Finder** | Пользователь | Описываешь что хранить текстом → AI (Claude) рекомендует размер бокса с объяснением. Без звонков и примерок |
| **Онлайн-бронирование** | Пользователь | Выбрал бокс → указал даты → отправил заявку. Оператор подтверждает. Без звонков |
| **Личный кабинет** | Пользователь | Мои бронирования, избранные склады, профиль, история |
| **Панель оператора** | Оператор | Dashboard с заполняемостью, управление складами и боксами, обработка бронирований, аналитика |
| **CRM для лидов** | Оператор | Отслеживание потенциальных клиентов, статусы, активности |
| **Админ-панель** | Администратор | Модерация складов, верификация операторов, управление пользователями |

### Ключевые технологии

- **AI Box Finder (Claude API):** Пользователь описывает вещи текстом, нейросеть рекомендует оптимальный размер бокса. Уникальная фича — ни у одного конкурента в регионе нет.
- **Геопоиск (PostGIS):** Поиск складов в радиусе от точки на карте. Работает с координатами, не с текстовыми адресами.
- **RAG-инфраструктура (pgvector):** Подготовлена база для AI-чата, который будет отвечать на вопросы на основе реальных данных о складах (v2).

## 7. ТЕХНИЧЕСКИЙ СТЕК

### Архитектура

```
┌─────────────────────────────────────────────────────┐
│                    FRONTEND                          │
│  Next.js 14 (App Router) + Tailwind CSS + shadcn/ui │
│  React Query + Zustand                               │

└────────────────────┬────────────────────────────────┘
                     │ Cookie-based JWT (httpOnly)
                     ▼
┌─────────────────────────────────────────────────────┐
│                    BACKEND API                       │
│  NestJS 10 (Monolith) + Prisma 5 ORM                │
│  Deployed: Railway                                    │
│                                                       │
│  Modules:                                             │
│  ├─ Auth (JWT cookies, register/login/refresh)       │
│  ├─ Users + Operators                                 │
│  ├─ Warehouses (PostGIS geo search)                  │
│  ├─ Boxes (inventory management)                     │
│  ├─ Bookings (state machine)                         │
│  ├─ Reviews + Favorites                              │
│  ├─ CRM (leads, contacts, activities)                │
│  ├─ AI Box Finder (Claude API)                       │
│  ├─ Media (S3 file upload)                           │
│  ├─ Notifications (SendGrid + Twilio)                │
│  ├─ Admin (moderation, verification)                 │
│  └─ Event Bus → Activity Log + Search Log            │
└────────────────────┬────────────────────────────────┘
                     │
          ┌──────────┼──────────┐
          ▼          ▼          ▼
   ┌──────────┐ ┌────────┐ ┌──────┐
   │PostgreSQL│ │ Redis  │ │ AWS  │
   │ +PostGIS │ │   7    │ │  S3  │
   │+pgvector │ │        │ │      │
   └──────────┘ └────────┘ └──────┘
```

### Ключевые технические решения

| Решение | Почему |
|---------|--------|
| **NestJS монолит** | Быстрый запуск. Микросервисы — после Product/Market Fit |
| **Cookie-based JWT** | Безопаснее Bearer-токенов: httpOnly, secure, sameSite |
| **PostGIS** | Настоящий геопоиск (ST_DWithin), не текстовый search |
| **pgvector** | Инфраструктура для RAG-поиска по складам (AI Chat v2) |
| **Event Bus** | Все действия логируются для обучения AI и аналитики |
| **Railway + Vercel** | Минимальные расходы на MVP. Миграция на AWS при росте |

### Общие вопросы

**Q: Что такое StorageCompare?**
A: Это онлайн-платформа для поиска и бронирования складских боксов в ОАЭ. Как Booking.com, только для складов. Собираем все предложения в одном месте, чтобы можно было сравнить цены и забронировать за минуту.

**Q: Чем отличается от обычного поиска в Google?**
A: Google покажет список складов — без цен, без свободных мест, без возможности забронировать. StorageCompare показывает реальные цены, свободные боксы, фотографии, отзывы и позволяет забронировать онлайн.

**Q: Это бесплатно для пользователей?**
A: Да, полностью. Поиск, сравнение, бронирование — бесплатно. Платит оператор (владелец склада) комиссию с бронирования.

**Q: Как работает AI Box Finder?**
A: Описываешь свои вещи текстом (например: «диван, стиральная машина, 15 коробок с книгами»), и AI анализирует объём и рекомендует оптимальный размер бокса с объяснением, почему именно этот. Больше не нужно угадывать или переплачивать за лишние метры.

**Q: В каких эмиратах работает?**
A: Запуск в Дубае и Абу-Даби. Планируем расширение на Шарджу, Аджман и все эмираты.

### Вопросы от операторов

**Q: Сколько стоит размещение на платформе?**
A: Размещение бесплатно. Комиссия 12% — только с подтверждённых бронирований. Нет бронирований — нет расходов.

**Q: Как стать оператором?**
A: Регистрация на сайте → заполнение профиля и документов → верификация администратором → размещение складов и боксов → приём бронирований.

**Q: Нужен ли свой сайт?**
A: Нет. Панель оператора на StorageCompare полностью заменяет сайт: управление складами, боксами, бронированиями и аналитика — всё в одном месте.

### Технические вопросы

**Q: Какие данные о складе нужно предоставить?**
A: Название, адрес, описание, фото, рабочие часы, характеристики (охрана, климат-контроль, 24/7 доступ). Для каждого бокса: размер (м²), цена, статус.

**Q: Как происходит оплата?**
A: В MVP оплата офлайн — пользователь бронирует на платформе, оплачивает напрямую оператору. Онлайн-оплата планируется в v2.

**Q: Безопасны ли данные?**
A: Все данные хранятся на серверах в регионе ОАЭ (AWS Bahrain). Авторизация через защищённые httpOnly cookies. Пароли хэшируются. Соответствие PDPL (закон о защите данных ОАЭ).
